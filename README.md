# String_in_python

# Python Practice Set - (Strings)

## Overview

This repository contains solutions to basic Python string manipulation exercises. These programs are designed to help beginners understand user input, string formatting, string methods, and escape sequence characters in Python.

## Programs Included

### 1. Greeting User with Good Afternoon

A Python program that takes a user's name as input and displays a personalized greeting message followed by "Good Afternoon".

**Concepts Used:**

* `input()` function
* String concatenation
* User interaction

### 2. Letter Template Formatter

A program that fills a predefined letter template by replacing placeholders such as name and date with user-provided values.

**Concepts Used:**

* String replacement
* Template formatting
* Variables

### 3. Double Space Detector

A Python program that checks whether a string contains double spaces and identifies their position.

**Concepts Used:**

* String searching
* `find()` method
* Conditional statements

### 4. Double Space Remover

A program that replaces all double spaces in a string with single spaces to improve text formatting.

**Concepts Used:**

* String manipulation
* replace() method

### 5. Letter Formatting Using Escape Sequences

A program that formats a letter using escape sequence characters such as newline (`\n`) and tab (`\t`) for better readability.

**Concepts Used:**

* Escape sequence characters
* Text formatting
* String output

## Learning Outcomes

Through these exercises, I learned:

* How to take user input in Python.
* Working with string methods and operations.
* Formatting text using escape sequences.
* Searching and replacing characters within strings.
* Creating dynamic and user-friendly programs.
